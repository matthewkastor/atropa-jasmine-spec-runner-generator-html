Class Detail
JasmineSpecRunnerGenerator(jasmineDirectory, outputFile, pageTitle)
Generates an HTML Jasmine Spec Runner for running tests in a web browser. 
var JasmineSpecRunnerGenerator = require(
     'atropa-jasmine-spec-runner-generator-html');

specRunner = new JasmineSpecRunnerGenerator(
     '/js/jasmine/', 'banana tester.html', 'Banana Test Suite');

specRunner.addSources(__dirname, '/js/myLib/');

specRunner.addSpecs('./specs/', '/myLib/test/specs/');

specRunner.generate(console.log);


// specRunner.generateFile(
//     function () {
//         console.log('ok');
//     }
// );
Parameters:
{String} jasmineDirectory
Optional. The location of jasmine-standalone relative to your web root. Defaults to /jasmine/
{String} outputFile
Optional. The name and location of the output HTML Jasmine Spec Runner. Defaults to Jasmine Spec Runner.html
{String} pageTitle
Optional. The title of the output page. Defaults to Jasmine Spec Runner.
